This is the data I get from the the link: https://grouplens.org/datasets/movielens/. It includes about 1000 users that rates from 0 to 5 of the movie “Toy Story”. Users are maked with gender Male and Female, which we can calculate the average rating score of each gender. So that we can figure out whether male like this move more or female.

We can tell from the image that the rating of both gender is close, whcih is close to 4. This means that this movie is popular in both gender and there is no extend difference between them. And the overall rating of this movie is high.
